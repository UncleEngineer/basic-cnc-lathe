📋 คู่มือการเขียน G-Code สไตล์ Heidenhain

วิเคราะห์จากไฟล์ .hnc จริง 14 ไฟล์ — ครอบคลุมโครงสร้าง, Peck Drilling และ Pattern ที่ใช้บ่อย

โครงสร้างโปรแกรม ส่วน Header กำหนด Tool ลำดับ Start-up Peck Drilling M/F/R Code Template พร้อมใช้ กฎสำคัญ

1. โครงสร้างโปรแกรมโดยรวม

ทุกไฟล์ .hnc มีโครงสร้างเดียวกัน 5 ส่วนหลัก ตามลำดับเสมอ

1
BEGIN / BLK FORM — ประกาศโปรแกรม & กำหนดชิ้นงาน
บรรทัด 0–2: ชื่อโปรแกรม, ขนาด Workpiece
2
TOOL DEF / TOOL CALL — กำหนดและเรียกใช้ดอกเจาะ
บรรทัด 3–4: นิยาม Radius + เรียก Spindle
3
Start-up Sequence — ยก Z, เปิด Spindle, เปิด Coolant
บรรทัด 5–7: ลำดับ Start ที่ตายตัวเสมอ
4
Drilling Loop — วนเจาะแต่ละรู
บรรทัด 10+: เคลื่อนไปรู → Peck → Retract → รูถัดไป
5
End Sequence — ยก Z กลับ, จบโปรแกรม
L Z+50 R0 F10000 M2 แล้ว END PGM

2. ส่วน Header — บรรทัด 0 ถึง 2

ตัวอย่างจากไฟล์ 7227.hnc Header
0BEGIN PGM 7227 MM
1BLK FORM 0.1 Z X+0 Y+0 Z-50
2BLK FORM 0.2 X+100 Y+100 Z+0
คำสั่งความหมายหมายเหตุ
BEGIN PGM [N] MMเริ่มโปรแกรม ชื่อ N หน่วยมิลลิเมตรN = ชื่อไฟล์ เช่น 7227
BLK FORM 0.1 Z X+0 Y+0 Z-50มุมล่าง-ซ้าย ของ Workpiece simulationZ-50 = ความลึกชิ้นงาน 50mm
BLK FORM 0.2 X+100 Y+100 Z+0มุมบน-ขวา ของ WorkpieceZ+0 = ผิวหน้าชิ้นงาน
หมายเหตุ: BLK FORM ใช้สำหรับ Simulation เท่านั้น ค่า X+100 Y+100 Z-50 ใช้เป็น template คงที่ในทุกไฟล์ ไม่ต้องเปลี่ยน

3. กำหนดและเรียกใช้ Tool — บรรทัด 3 ถึง 4

Tool Definition & Call Tool Setup
3TOOL DEF 1 L+0 R+3
4TOOL CALL 1 Z S2000
พารามิเตอร์ความหมายค่าที่พบ
TOOL DEF 1นิยาม Tool หมายเลข 1ใช้ Tool 1 เสมอ
L+0Length offset ของดอก+0 ทุกไฟล์ (คงที่)
R+[n]รัศมีดอกเจาะ (mm)เปลี่ยนตามดอก: 2.25, 2.5, 3, 3.5, 3.75, 4, 5
TOOL CALL 1 Z S2000เรียก Tool 1 แกน Z รอบ 2000 RPMคงที่ทุกไฟล์

Radius → ขนาดดอกเจาะ

  • R+2.25 → Ø 4.5mm
  • R+2.5 → Ø 5mm
  • R+3 → Ø 6mm
  • R+3.5 → Ø 7mm
  • R+3.75 → Ø 7.5mm
  • R+4 → Ø 8mm
  • R+5 → Ø 10mm

กฎการตั้งค่า Tool

  • ใช้ Tool หมายเลข 1 เสมอ (1 Tool/โปรแกรม)
  • L+0 คงที่ ไม่เปลี่ยน
  • S2000 คงที่ ไม่เปลี่ยน
  • R ต้องตรงกับดอกเจาะจริง

4. ลำดับ Start-up — บรรทัด 5 ถึง 7

3 บรรทัดนี้เหมือนกัน ทุกไฟล์ ห้ามเปลี่ยน

Start-up Sequence (ตายตัว) Always Same
5L Z+50 R0 F10000 M3
6L X-2 Y-2 F10000
7L Z+50 F1000 M8
บรรทัดคำสั่งความหมาย
5L Z+50 R0 F10000 M3ยก Z+50, ยกเลิก Radius comp (R0), ฟีดเร็ว, เปิด Spindle CW (M3)
6L X-2 Y-2 F10000เคลื่อนไปตำแหน่งนอกชิ้นงาน X-2 Y-2
7L Z+50 F1000 M8ยืนยัน Z+50 ด้วย F1000, เปิด Coolant (M8)
เหตุผล X-2 Y-2: ให้สปินเดิลหมุนเสถียรก่อนเข้าชิ้นงาน เป็น Safe position ก่อนเริ่มเจาะ

การนับบรรทัด (Line Numbers)

บรรทัด 0–7 นับเรียง 1 ต่อ 1 จากนั้น กระโดดไปบรรทัด 10 แล้วนับเพิ่มทีละ 2 ตลอดโปรแกรม

Pattern การนับบรรทัด
0–7← Header + Tool + Start-up (นับทีละ 1)
10← รูแรก (กระโดดข้าม 8, 9)
12 14 16 18 ...← นับทีละ 2 ตลอด

5. Pattern การเจาะ (Peck Drilling)

มี 2 รูปแบบขึ้นอยู่กับ ความลึก ของรูเจาะ

5.1 เจาะรูตื้น (Single Peck) — ความลึกไม่เกิน ~5mm

Single Peck — ตัวอย่าง ลึก -3mm ตื้น
10L X-312.000 Y0.000 F1000 ← เคลื่อนไปตำแหน่ง XY รู (อยู่ที่ safe height)
12L Z30.000 F ← ลง Clearance Z30
14L Z3.000 F ← Approach ใกล้ผิว Z+3
16L Z-3.000 F80 ← เจาะ -3mm ด้วย F80 (Cutting feed)
18L Z30.000 F ← Retract → รูถัดไป

แผนภาพ Z — Single Peck

Z +50Safe height
Z +30Clearance height (ระหว่างรู)
Z +3Approach ก่อนเจาะ
Z -3↓ Cutting depth F80
Z +30Retract → รูถัดไป

6. M-Code, F-Code และ R-Code

M-Code ที่ใช้

Codeความหมาย
M3เปิด Spindle หมุน CW
M8เปิด Coolant
M2จบโปรแกรม

F-Code (Feed Rate)

ค่าใช้เมื่อ
F10000Rapid (ยก Z, เปิดโปรแกรม)
F1000เคลื่อน XY ไปรู
F80Cutting (ดอก Ø6 ขึ้นไป)
F70Cutting (ดอก Ø < 6mm)
F (ว่าง)ใช้ Feed ล่าสุด

R-Code (Radius Comp)

Codeใช้ที่ไหน
R0บรรทัด 5 และบรรทัดสุดท้าย เท่านั้น
(ไม่ใส่)ทุก L ระหว่างเจาะ ไม่ใส่ R เลย
F ว่าง: L Z30.000 F คือใช้ Feed rate จากบรรทัดก่อนหน้า ไม่ต้องพิมพ์ซ้ำ แต่ต้องรู้ว่า Feed ล่าสุดคือค่าใด

8. กฎสำคัญ (สรุป)

✓ ต้องทำเสมอ

  • BEGIN/END PGM ใส่เลขตรงกับชื่อไฟล์
  • BLK FORM 0.1 และ 0.2 ทุกโปรแกรม
  • Start-up 3 บรรทัด (5–7) ตายตัว
  • กระโดดจาก 7 → 10 นับ +2 ตลอด
  • จบด้วย L Z+50 R0 F10000 M2
  • XY ใส่ทศนิยม 3 ตำแหน่ง: X-312.000

⚡ Peck Rules

  • Approach ที่ Z+3 ก่อนเจาะทุกครั้ง
  • Cutting: F80 (ดอกใหญ่) / F70 (ดอกเล็ก)
  • Peck แรก ใส่ F เสมอ
  • Retract ระหว่าง Peck = Z+3 (ไม่ใช่ Z10/Z30)
  • ระหว่างรู: retract Z10 หรือ Z30
  • Multi Peck: เพิ่มความลึกทีละ 5mm

🚫 ห้ามทำ

  • ห้ามเปลี่ยน S — ใช้ S2000 เสมอ
  • ห้ามใส่ Radius Comp ระหว่างเจาะ
  • R0 มีเฉพาะบรรทัด 5 และสุดท้าย
  • ห้ามเคลื่อน X/Y ขณะ Z อยู่ในชิ้นงาน
